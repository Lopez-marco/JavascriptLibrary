Borders

the border property allows you to specify the style, width, and color od an elemnet's boder.

Border style

The border-style property can take up to four values (top border. right border, bottom border. and left border). the following border-style values are alloed:

  • dotter:dotted border
  • dashed:dashed border
  • solid:solid border
  • double:double border
  • groove:3d groove border, depends on border-color value
  • ridge:3d ridge border, depends on the border-color value
  • insert:3d insert border, depends on the border-color value
  • none:no border
  • hidden:hidden border
  • mixed:you can set a differnt border style for the top, bottom, right, and left.

Border width

VERY IMPORTANT: border-width will not work unless you have border-style set!!

Border-width can be set to a specific size (px, em, etc) or you can use one of the following pre-set settings: thin, medium, thick. The border-width property can also take up to four values (top, right, botom, left)

Border color

VERY IMPORTANT: border-color will not work unless you have border-style set!!

border-color can be set by the following ways:

  • name:specify a color as "blue"
  • Hex:specify a hex value, like #eee
  • RGB:specify a RGB value, like RGB(100,100,100)
